Tree and Ensemble Learning

Classification via regression trees and neural networking

This paper examines the classification of abalone age by specific physical measurements, as well as the categorical variable of sex, into four different age classes. Decision trees with pre-pruning and post-pruning, ensemble methods such as bagging and pasting, as well as neural network models were employed to examine this question.

A GridSearchCV was used to find the optimal neural network hyperparameters.

The full paper can be found here and the JupyterLab notebook can be found here.

Neural Networks

Implementing different neural networks

This paper examines:

  • comparison accuracy of the classification of the Kuzushiji Japanese character set via NetLin, NetFull, NetConv and ResNet neural networks. The full code is here and here.
  • unwrapping the twin spirals using the PolarNet and RawNet neural networks, optimizing the number of nodes necessary to achieve consistent results. The full code can be found here and here.
  • an exploration of Hidden Unit Dynamics, in this case, generating fun tensor art of the “crab” character from Space Invaders and Yoda from Star Wars. The full code can be found here and here.

Natural language Processing

This paper describes the use of a 1-, 2- and 3-ngram approach to natural language processing to undertake both sentiment analysis and business characterization. The 2000 optimal ngrams were chosen from the dataset, followed by a 285-node hidden layer, followed by a 40-node hidden layer, which produced the categorization information, followed by a last layer to determine the sentiment. The code for this project can be found here.

Regression projects

Australian Energy Market Operator (AEMO)

This work was the CAPSTONE project for my Master’s degree in DataScience at UNSW. It worked with the Australian Energy Market Operator (AEMO), to be able to predict the different Australian states’ energy demands at different future moments. While this was a team project, each individual member was assigned different duties; my duties included the linear model (LM) as well as the generalized linear model (GLM). It used “roll forward analysis”, where all the previous data is used to train various models to make predictions about future energy needs.

My section of the code is here, as well as the full team GitHub project, the project report and the project video.

Machine learning paper: Linear Regression

The purpose of this paper is to use linear models, including regression, to predict the age of abalone from supplied physical measurements, and the classification of abalone sex - generating heatmaps, scatterplots, categorical plots, and histograms. It undertook an Exploratory Data Analysis (EDA), correlation analysis, and various types of modelling and regression.

Natural Language Processing

Ngrams for business categorization and sentiment analysis

This paper describes the use of a 1-, 2- and 3-ngram approach to natural language processing to undertake both sentiment analysis and business characterization. The 2000 optimal ngrams were chosen from the dataset, followed by a 285-node hidden layer, followed by a 40-node hidden layer, which produced the categorization information, followed by a last layer to determine the sentiment. The code for this project can be found here.
